
# Performance Review Process

## Review Cycle
- Annual reviews: December
- Mid-year check-ins: June
- Quarterly 1-on-1s with manager

## Performance Ratings
1. Needs Improvement
2. Meets Expectations
3. Exceeds Expectations
4. Outstanding
5. Exceptional

## Components
- Goal achievement (40%)
- Competencies (30%)
- Leadership/collaboration (20%)
- Innovation/initiative (10%)
